Deprecations
------------

* Using :meth:`.Dataset.astype` as a context manager (``with dset.astype(t):``)
  is deprecated. Slice the object returned by astype instead
  (``data = dset.astype(t)[:10]``). This works from h5py 3.0 onwards.
